# Pytest Automation Framework (POM Structure)

## Project Overview
This is a simple Automation Testing Framework built using **Python, Pytest, and Page Object Model (POM)**.  
It is designed for web application testing and helps in maintaining clean, reusable, and scalable test scripts.

---

## Technologies Used
- Python 3.x
- Pytest
- Selenium WebDriver (if used)
- Page Object Model (POM)
- VS Code
